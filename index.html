18 Vertical - Make some rythm to your text.

18Vertical is a mini stylesheet that turn the standar typography of broswer to pretty well formated text.

Some lists.

Classic style.

  • First element.
  • Second element.
  • Third element.

CODE EXAMPLE

<ul>
  <li>First element.</li>
  <li>Second element.</li>
  <li>Third element.</li>
</ul>

With some order.

  1. First element.
  2. Second element.
  3. Third element.

CODE EXAMPLE

<ol>
  <li>First element.</li>
  <li>Second element.</li>
  <li>Third element.</li>
</ol>
